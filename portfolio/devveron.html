Devveron Text Based Adventure Application/Online Chatroom

In my final bootcamp group project, we built a text based adventure game/online chat room application using mostly socket.io and React libraries.

My role and responsibilities included:

  • Connecting and building up the chain of CRUD operations from the backend to the frontend (i.e. database, Express routes, API client, React components)
  • Wire Framing/Storyboarding/User Stories
  • Project Planning and Content Creation
  • Scrum Lead for the group ie. Maintaining Agile principles such as taking charge in progressing stories on our kanban board, scheduling standups and making sure we keep to our allotted time boxes more generally.
