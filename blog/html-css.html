What is the difference between Margin, Border, and Padding?

The main difference between padding and margin is that padding controls how much breathing room exists within a box, while margin controls how much breathing room or whitespace exists outside of a box. Borders represents the edge of a CSS box which extends around the pudding and the content, the margin is 0, being invisible, but it is possible to set the style, thickness or color of the border.

What are the best practices associated with using classes vs. ids?

Class and Id are selectors used to organize styling elements within CSS code. The Class selector is used to identify and group multiple elements (ie class) whilst IDs on the other hand are unique to specific elements and as a general rule should only be used once

What are the differences between relative, absolute, and fixed positioning?

Static - this is the default value, all elements are in order as they appear in the document.

Relative - the element is positioned relative to its normal position.

Absolute - the element is positioned absolutely to its first positioned parent.

Fixed - the element is positioned related to the browser window.

What does it mean to display inline vs inline blocks?

display: block means that the element is displayed as a block, as paragraphs and headers have always been. A block has some whitespace above and below it and tolerates no HTML elements next to it, except when ordered otherwise (by adding a float declaration to another element, for instance).

display: inline means that the element is displayed inline, inside the current block on the same line. Only when it's between two blocks does the element form an 'anonymous block', that however has the smallest possible width.
